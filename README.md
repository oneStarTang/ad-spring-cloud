# ad-spring-cloud 基于springcloud搭建的微服务广告系统

## 主要包含：
- 广告主投放广告的《广告投放系统》 
- 媒体方（广告展示媒介-<地铁广告屏幕>）
- 检索广告用的《广告检索系统》 
- 广告计费系统（按次，曝光量等等） 
- 报表系统
- Etc.
  
## 使用技能栈
- JDK1.8 
- MySQL 8+ 
- Maven 3+ 
- Spring cloud F 
- Eureka
- Zuul / gateway 
- Feign ... 
- Spring boot 2.x 
- Kafka 2.2.0 
- MySQL 
- Binlog
